🔌 One-time setup

Connect a free shared backend so everyone's bets sync in real time.

This app needs a tiny free database so your phone, and your friends' phones, all see the same live league. Takes about 5 minutes, no credit card.

  1. Go to console.firebase.google.com → Add project → give it any name → skip Google Analytics → Create.
  2. Left sidebar → Build → Realtime Database → Create Database → pick any location → start in test mode.
  3. Click the Rules tab and replace the contents with: 
    {
  "rules": { ".read": true, ".write": true }
}
    then Publish. (This keeps it simple for a private friend group — don't share the link publicly.)
  4. Click the ⚙️ gear next to Project Overview → Project settings → scroll to Your apps → click </> (web) → register the app → copy the firebaseConfig object it shows you.
  5. Open this HTML file in a text editor, find FIREBASE_CONFIG near the top of the <script> section, and paste your 6 values in. Save the file and re-upload it to wherever you're hosting it.

Just exploring? You can also open this file locally without any setup — but bets won't sync between different phones until this step is done.

❓ How to Play

📏 The Rules

  • Every fight night, each player gets a fresh 1000 points to bet.
  • Bets come in three sizes only: 20, 60, or 100 points.
  • Bet on as many fights as you like — total staked can't exceed 1000.
  • One fighter per fight. You can't hedge by betting both corners.
  • Points you don't bet are safe and stay in your bankroll.
  • ⏰ Betting locks 30 minutes before each fight starts. Once a fight locks, your saved bet on it stands — no changes.
  • Your final bankroll for the night gets added to your season total.

📡 Fight cards and start times sync automatically from ESPN. Odds sync from a live sportsbook feed (see the "Odds Source Setup" card below). Hit "Sync now" in 🥊 Events to refresh manually.

🔌 Odds Source Setup

Odds come from The Odds API — real lines from DraftKings, FanDuel, and others, aggregated into one feed. Free, no card required:

  1. Go to the-odds-api.com → sign up for a free API key.
  2. Open this HTML file in a text editor, find ODDS_API_KEY near the top of the script, and paste your key in.
  3. Save and re-upload the file — odds will start filling in on the next sync.

Free tier covers 500 lookups/month, plenty for a weekly league. Without a key, fights just show −110/−110 until you edit odds manually in 🥊 Events.

📐 How Payouts Work

We use real American odds, converted to a multiplier:

  • Underdog (+250): multiplier = 1 + 250/100 = 3.50×
  • Favorite (−180): multiplier = 1 + 100/180 = 1.56×

Win: you get back stake × multiplier. Lose: the stake is gone.

🧮 Worked Example

You start with 1000 and place two bets:

  • 100 on a +400 underdog → he wins! You get back 100 × 5.00 = 500 🔥
  • 60 on a −200 favorite → he loses. Your 60 is gone.

Final bankroll = 1000 − 160 staked + 500 returned = 1340 added to your season total.
